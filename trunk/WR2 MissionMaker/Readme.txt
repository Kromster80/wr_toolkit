------------------------------------------------
World Racing 2 Mission Maker - Beta
------------------------------------------------

Description:
This tool lets you make new missions for WR2

Key Features:
- Random mission maker

Requirements:
- MissionMaker must be run in WR2 folder.
- backup "FrontEnd\WR.ds" file, just in case.
- Missions should be installed by WR2Manager

Interface:
- So far so good

Known Bugs:
This is a Beta. There should be no major bugs. If you find one please let me know.
