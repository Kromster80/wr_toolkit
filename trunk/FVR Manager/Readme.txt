     <<Ferrari Virtual Race>> Manager by Krom v0.1b

Description:
This tool installs and manages FVR add-on cars.

Key Features:
- Add-on car installer :-)

Requirements:
- Manager must be run from FVR folder.

Manager does the following:
- Changes installed cars info in FVR database "FrontEnd.ds" file.
